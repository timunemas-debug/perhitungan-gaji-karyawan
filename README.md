# perhitungan-gaji-karyawan
program sederhana untuk menghitung gaji karyawan berdasarkan golongan, pendidikan, dan jumlah jam lembur
## Fitur
- Input data karyawan
- Hitung tunjangan golongan dan pendidikan
- Hitung upah lembur
- Cetak rincian gaji lengkap

## Cara Menjalankan
```bash
python perhitungan_gaji_karyawan.py
